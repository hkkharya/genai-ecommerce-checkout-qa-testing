# GenAI-Assisted E-Commerce Checkout QA Testing & Process Improvement Project

## Project Overview

This project demonstrates end-to-end manual QA testing and process improvement using the nopCommerce Demo Store.

The project focuses on validating core e-commerce functionality, creating test artifacts, identifying usability observations, and recommending customer experience improvements with support from GenAI.

GenAI was used to enhance requirement analysis, test scenario generation, documentation quality, and process improvement recommendations, while all testing activities and validations were performed manually.

---

## Application Under Test

**Application Name:** nopCommerce Demo Store

**Website:** https://demo.nopcommerce.com

---

## Project Objectives

- Perform end-to-end manual testing of the checkout workflow.
- Create requirements, test scenarios, and detailed test cases.
- Execute tests and document actual results.
- Identify usability observations and process improvement opportunities.
- Use GenAI to support requirement analysis, test design, and documentation.
- Establish requirement traceability using RTM.
- Summarize execution metrics through dashboard reporting.
